Quick Start: Your First Animation

  1. Use the Pencil tool and Brush levels to draw on the heatmap. This is your first frame.
  2. In the "Frames" panel on the right, click the Duplicate Frame button (üìÑ).
  3. A new frame appears, selected. Make a small change to your drawing on the heatmap.
  4. Repeat steps 2 and 3 a few times to create more frames.
  5. In the "Animation" panel, click the Play button (‚ñ∂Ô∏è) to see your animation!
  6. Adjust the FPS slider to change the speed.

Left Panel Explanations

View

  • 100% / 150% / 200%: Sets a fixed zoom level for the drawing canvas.
  • Fit Width: Automatically resizes the drawing canvas to fit the available horizontal space.

Data

This section handles data for the entire project. The JSON format is preferred for saving and loading your work as it includes all frames, layers, and settings.

  • Import/Export JSON: Saves or loads the entire state of your project, including all frames, layers, colors, and thresholds. Use this for complete backups or transferring your work.
  • Import/Export CSV: Imports or exports a simple `date,count` format. This only affects the currently active layer and is useful for transferring contribution data from other sources like spreadsheets.

Right Panel Explanations

Animation

  • FPS: Frames Per Second. Higher is faster.
  • Onion Skin: Shows a faint outline of the previous frame on the canvas to help with animation.
  • Stable Months: Keeps the month labels (Jan, Feb, etc.) fixed to the first frame's dates during animation. Uncheck for scrolling month labels.

Frames & Layers

An animation is made of Frames. Each frame can have multiple Layers, though most animations only need one layer.

  • [+]: New Blank Frame.
  • [üìÑ]: Duplicate the currently selected frame.
  • [+üñºÔ∏è]: Adds the currently active *layer* as a new frame. Useful for building animations from multiple layers.
  • Shift Arrows (↑↓←→): Move the entire content of the current frame. This is powerful when combined with the options below:
    • Wrap: When checked, any content moved off one edge of the canvas will wrap around and reappear on the opposite side.
    • Auto Wrap Animation: When checked, pressing a shift arrow will automatically generate all the necessary frames to create a full, looping animation of the content scrolling across the canvas and returning to its starting position. This is perfect for creating smooth scrolling backgrounds or marquees from a single drawing.

Layer Data

  • Import / Export Layer: This opens a text box with the JSON data for only the single, currently active layer. This is different from the main Data panel; use this to copy/paste a specific drawing between projects.

Export Animation

  • Current as PNG/WEBP: Saves a static image of what you currently see on the active layer.
  • Animation (PNGs): Exports every frame of your animation as a separate numbered PNG file.
  • Animation (GIF): Combines all frames into a single, looping animated GIF file. This is a classic format that is widely supported.
  • Animation (WEBP): DOES NOT CURRENTLY FUNCTION! Combines all frames into a single, looping animated WEBP file. This is a modern format that offers higher quality and smaller file sizes than GIF, but may not be supported on older websites or apps.
  • Include Labels: When checked, the exported image(s) or animation will include the day and month labels. Uncheck to export only the pixel grid.
